# Temporary Review Notes

This folder contains branch-review and handoff material that is useful while
the `wei-han` branch is being prepared, but should be deleted before final
course submission if the repository is submitted as a polished artifact.

These files are not required to run the dashboard, execute tests, understand
the architecture, or map the repository to the QF635 report guidelines.

Current contents:

| File | Purpose |
|------|---------|
| `BRANCH_CHANGES.md` | Human-readable log of intentional branch changes from `main` |
| `SPLIT_AUDIT.md` | Internal maintainability audit for possible future module splits |
| `SUBMISSION_HANDOFF.md` | Temporary checklist for teammates before final submission |

Before final submission, either delete this folder or move any still-useful
content into the official documentation under `docs/`.
